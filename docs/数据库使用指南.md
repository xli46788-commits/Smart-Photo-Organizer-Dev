# 本地数据库使用指南

## 连接信息（与 application.yml 一致）

| 项目 | 值 |
|------|-----|
| 主机 | `localhost` |
| 端口 | `3306` |
| 数据库名 | `imprint` |
| 用户名 | `root` |
| 密码 | **安装 MySQL 时你自己设置的密码**（不一定是 root） |

> 若后端启动报 `Access denied`，请把 `backend/src/main/resources/application.yml` 里的 `password` 改成你的真实 MySQL 密码。

---

## 方法一：命令行打开（最快）

### 1. 打开 PowerShell 或 CMD

### 2. 登录 MySQL

```powershell
& "D:\MySQL\MySQL Sever8.0\bin\mysql.exe" -u root -p
```

回车后**输入你的 MySQL 密码**（输入时不会显示字符，正常现象）。

### 3. 常用命令

```sql
-- 查看所有数据库
SHOW DATABASES;

-- 进入本项目数据库（需先执行 sql/init.sql 初始化）
USE imprint;

-- 查看表
SHOW TABLES;

-- 查看用户
SELECT * FROM user;

-- 查看照片
SELECT id, original_name, ai_category FROM photo;
```

### 4. 首次初始化数据库

登录 MySQL 后执行：

```sql
source D:/Users/lzl/Desktop/智能照片整理应用开发/sql/init.sql
```

或在 PowerShell 一行命令（把 `你的密码` 换成真实密码）：

```powershell
& "D:\MySQL\MySQL Sever8.0\bin\mysql.exe" -u root -p你的密码 -e "source D:/Users/lzl/Desktop/智能照片整理应用开发/sql/init.sql"
```

## 方法二：图形界面（推荐新手）

任选一种免费工具，新建连接时填写上面的连接信息。

### DBeaver（推荐，免费）

1. 下载：https://dbeaver.io/download/
2. 安装后打开 → **数据库** → **新建连接** → 选 **MySQL**
3. 填写：
   - Host: `localhost`
   - Port: `3306`
   - Database: `imprint`
   - Username: `root`
   - Password: 你的 MySQL 密码
4. 测试连接 → 完成

### MySQL Workbench（MySQL 官方）

1. 下载：https://dev.mysql.com/downloads/workbench/
2. 打开 → **MySQL Connections** → **+** 新建
3. Connection Name: `IMPrint本地`
4. Hostname: `127.0.0.1`，Port: `3306`，Username: `root`
5. 点 **Store in Vault** 保存密码 → Test Connection → OK

### Navicat / HeidiSQL

同样新建 MySQL 连接，主机 `localhost`，端口 `3306`，用户 `root`，密码填你的即可。

## MySQL 服务未启动时

1. `Win + R` → 输入 `services.msc` → 回车
2. 找到 **MySQL80** → 右键 → **启动**

或以**管理员** PowerShell：

```powershell
Start-Service MySQL80
```

---

## 忘记 root 密码？

1. 以管理员打开 CMD，停止服务：`net stop MySQL80`
2. 跳过权限启动：`mysqld --console --skip-grant-tables --shared-memory`（另开窗口）
3. 无密码登录：`mysql -u root`
4. 重置密码：
   ```sql
   FLUSH PRIVILEGES;
   ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';
   ```
5. 关闭 mysqld，再 `net start MySQL80`
6. 同步修改 `application.yml` 中的 `password`

---

## 验证后端能否连上数据库

密码配置正确后，在项目目录：

```powershell
cd backend
mvn spring-boot:run
```

看到 `Started ImprintApplication` 且无数据库报错，说明连接成功。
